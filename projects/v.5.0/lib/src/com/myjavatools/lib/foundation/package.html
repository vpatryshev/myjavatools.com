com.myjavatools.lib.foundation deals with very general notions: Function, Filter (Predicate), and their applications to Collections framework classes.

Package Specification

Function<X,Y> is a holder for a method mapping X to Y. A function can be easily build from a Map; a Map can be build from a function if you provide a Set of keys.
Having a function: X -> Y, you can create a List<Y> view of function values on a List<X>
Filter<T> is a Function<T,Boolean>, and can be used for filtering (grepping) collections, iterables, etc.
Maps contains a loose group of static methods that are responsible for standard operations on maps: composition, reverse, restriction to a collection, application of Maps to Lists, Iterables, factory methods that create maps from key/value pairs.
Iterators contains several methods for iterators: concatenation, applying a function to an iterator, grepping an iterator using a filter.
